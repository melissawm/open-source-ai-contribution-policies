# Open source vs. IA: Um panorama das políticas de engajamento atuais

Categoria: FLOSS
Nível: Iniciante
Nível de conhecimento em Open Source: Básico

## Descrição

Com o uso cada vez mais frequente de IA por pessoas desenvolvedoras, um desafio recente entre mantenedores de projetos open source tem sido lidar com contribuições geradas total ou parcialmente por IA. Em alguns casos, a contribuição é totalmente automatizada, sem que um humano sequer supervisione o resultado.

Vários projetos já se mobilizam para instituir políticas que definem limites para o uso de IA em seus repositórios. Em https://github.com/melissawm/open-source-ai-contribution-policies, faço uma coleção de tais políticas, variando entre permissividade e proibição. 

Nesta palestra, vamos discutir o panorama atual, quais são as questões a serem consideradas e o que podemos esperar do futuro das contribuições em projetos open source.

## Principais atividades

O uso de ferramentas de IA, cada vez mais frequente, representa um grande desafio para projetos open source e redefine o que é uma contribuição aceitável.

## Como compilar os slides

Esses slides são feitos usando o [`sli.dev`](https://sli.dev).

Para criar os slides inicialmente:

```
pnpm create slidev
```

Os slides aparecerão em `http://localhost:3030/ `.

Para começar o slideshow:

```
pnpm install
pnpm run dev
```

O conteúdo dos slides está em [slides.md](./slides.md).
